Prostate Health Index (PHI) 文獻彙整

時間範圍：2020–2025（以 PubMed 為主，提供可靠摘要；Google Scholar 僅作延伸探索）

說明：本清單聚焦 PHI（Prostate Health Index）於攝護腺癌之診斷、風險分層、影像整合與經濟效益等主題。 多篇研究顯示 PHI 與其衍生指標（如 PHI density/PHID、%p2PSA）普遍優於單純 PSA 或 %fPSA，亦可輔助 MRI/PI‑RADS、PSMA 影像與積極監測決策。

  1. Prostate health index (PHI) as an accurate prostate cancer predictor（2023，J Cancer Res Clin Oncol）臨床

    連結：PubMed: 37204514
    Abstract
    前瞻性研究（n=140），比較 PHI 與 tPSA 預測攝護腺癌。PHI 辨識能力優於 tPSA（AUC：0.77 vs 0.44），加入 PHI 的多變量模型亦提升準確度。結論：PHI 在此族群優於 tPSA。

  2. PHI 作為可靠生物標記：系統回顧與統合分析（2022，Clin Chem Lab Med）統合分析

    連結：PubMed: 35567430
    Abstract
    納入 60 研究、14,255 人。整體 PCa 靈敏度/特異度 0.791/0.625；csPCa 為 0.874/0.569。結論：PHI 對 PCa 與 csPCa 具高準確性，可用於活檢決策輔助。

  3. PHI density（PHID）結合 MRI 在可疑影像之 csPCa 診斷優於 PSA density（2023，J Urol）多中心

    連結：PubMed: 37036248
    Abstract
    多中心（n=1,215），PI‑RADS + PHID 的 AUC 0.85，優於 PI‑RADS + PSAD（0.81）或單用 PHI（0.78）。在 PI‑RADS 3 病灶，PHID 可進一步減少不必要活檢。

  4. 初始 PHI/PHID 可預測初次陰性活檢後 6 年內風險（2022，Prostate Cancer Prostatic Dis）追蹤

    連結：PubMed: 34453109
    Abstract
    PSA 4–10 ng/mL、初次活檢陰性者（中位追蹤 77 個月），較高的基線 PHI/PHID 與後續 PCa 與高風險病灶相關。提出如 PHI ≥35、PHID ≥1.2 等分層門檻。

  5. PHI 用於初次活檢之成本效益（2020，Front Oncol）經濟性

    連結：PubMed: 33330035
    Abstract
    多中心決策模型（n=3,348）：PHI 門檻 23–35 可減少 22–50% 不必要活檢；建議整體門檻 27，具成本效益。

  6. 真實世界 2 年路徑：PHI 對臨床決策之影響（2025，BJU Int；2024 先行線上）實務

    連結：PubMed: 38961793
    Abstract
    香港公醫系統（n=2,828）：知悉 PHI 後，83% 升高 PSA 患者選擇暫不活檢；PHI ≥35 與 GG≥2 初次及後續偵測率較高。PHI 有助個人化活檢與追蹤頻率。

  7. PI‑RADS 3 病灶：PHI/PHID 之診斷價值（2023，Urol Int）PI‑RADS 3

    連結：PubMed: 38011845
    Abstract
    PHI（AUC 0.786）與 PHID（0.763）對 PCa 與 csPCa 均較傳統 PSA 指標佳。結論：PI‑RADS 3 案例中 PHI/PHID 具重要辨識力。

  8. 以 PHI 對 PI‑RADS 3 進行風險分層、降低活檢（2024，Asian J Androl）分層

    連結：PubMed: 37695241
    Abstract
    PI‑RADS 3 中，csPCa 偵測率隨 PHI 分層上升（如 PHI ≥55 為 52.6%）。以低 PHI 作為免活檢條件之代價較小。

  9. PHI 預測積極監測下之病理進展（2025，World J Urol）AS 監測

    連結：PubMed: 40650759
    Abstract
    低風險在監測者（n=77）：較高 PHI 與後續病理進展相關（AUC 0.641）。PHI 有助監測中之風險分層。

  10. PHI vs mpMRI 於活檢情境之比較（2021，World J Urol）比較

    連結：PubMed: 32761380
    Abstract
    395 人資料顯示 PHI AUC 69.7（PSA 59.5、PSAD 64.9、PIRADS 62.5）。PHI 加至 PIRADS 可顯著提升 csPCa 預測；適合用於 MRI/活檢前分流。

  11. PHI + 18F‑PSMA‑1007 PET/MR 之診斷價值（2024，Asian J Urol）影像整合

    連結：PubMed: 39534010
    Abstract
    前瞻性（n=41）：PET/MR 加成使 AUC 自 PHI 0.692 提至 0.802，並降低不必要活檢；但於 PHI ≥43.5 者增益有限。

  12. 以 PHI 作為 MRI 分流工具（2024，Clinical Biochemistry）MRI 分流

    連結：PubMed: 38583655
    Abstract
    懷疑 PCa 患者中，PHI 對預測 PI‑RADS ≥3 表現最佳（AUC 0.747）；最佳 PHI 臨界值約 30（敏感度 90%）。多變量下，僅 PHI 為獨立預測因子。

  13. PHI + mpMRI 預測積極監測下分級上調（2020，BJU Int）AS 監測

    連結：PubMed: 32367635
    Abstract
    AS 隊列（n=253）：在 PI‑RADS ≤3 者加入 PHI 臨界值後，NPV 提升至 98%，AUC 亦上升；約可避免 20% 活檢且僅少量遺漏升級。

  14. %p2PSA 與 PHI 優於 PSA/%fPSA，並可作為轉檢（reflex）條件（2022，Arch Pathol Lab Med）生物標記

    連結：PubMed: 34343237
    Abstract
    PSA 2–10 ng/mL（n=237）：%p2PSA（AUC 0.781）與 PHI（0.814）均優於 PSA 與 %fPSA。採 PHI ≥27 或 %p2PSA ≥1.25% 可省略 25–29% 活檢，約 10% csPCa 可能遺漏。

  15. PI‑RADS 3：PHI density（PHID）用於活檢策略（2024，JMRI）PI‑RADS 3

    連結：PubMed: 38299766
    Abstract
    活檢新手 PI‑RADS 3（n=193）：PHID AUC 0.793 最佳。設定兩個 PHID 臨界值可避免約 48% 活檢，約 5% csPCa 遺漏，且中間「灰區」比例最低。

  16. PHI/PHID 與 %fPSA 之診斷比較（2023，Transl Androl Urol）比較

    連結：PubMed: 37032752
    Abstract
    活檢族群（n=296）：PHID（AUC 0.880）與 PHI（0.867）皆優於 PSA 相關指標，於 PSA 4–10 ng/mL 灰區亦同。

  17. Beyond PSA：PHI 角色總覽（2020，Int J Mol Sci；綜述）綜述

    連結：PubMed: 32053990
    Abstract
    回顧血液與尿液多種標記；PHI 與具侵襲性 PCa 相關，並與 4Kscore、Stockholm3、尿液檢測等作比較。

  18. PHI 對積極監測中分級再分類之影響（2022，J Urol）AS 監測

    連結：PubMed: 35830553
    Abstract
    Canary PASS（1,142 名、1,532 次活檢）：於確認性活檢，加入 PHI 僅小幅提升 AUC（+0.021），於後續活檢無顯著增益。

  19. PHI 與 MRI 於活檢前之結合（2025，Int J Urol）影像整合

    連結：PubMed: 40098218
    Abstract
    n=105：PHI 對 PC 之 AUC 0.816（優於 PI‑RADS 0.753）。若僅對 PI‑RADS ≥4 或 PHI ≥38.1 施行活檢，可減少約 21% 活檢、僅極少數 csPCa 遺漏。

  20. 術前 PHI + 睪固酮預測根除手術後之生化復發（2023，Arch Esp Urol）術後預後

    連結：PubMed: 38186072
    Abstract
    RP 隊列（n=132）：PHI 與睪固酮為 BCR 獨立風險因子；兩者合併 AUC 0.906，較高 PHI 與較低睪固酮與較差無復發存活相關。

  7. Validation of PHI for cancer detection（2018）驗證

    連結：PubMed: 30041764

  9. Validation of PHI in a predictive model of prostate cancer（2018）模型

  10. PHI 為 mpMRI 之加值工具（重複活檢族群）（2016）影像加值

    Abstract
    於重複活檢族群，PHI 對 mpMRI 具增益效果，可提升顯著癌的偵測與決策品質。

  11. Elevated PHI 與活檢參數之關聯（2016）相關性

    Abstract

    連結：PubMed: 31356566
